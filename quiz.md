# LunchNear.me Personality Quiz

## Overview
A 7-question personality quiz that determines users' "lunch personality" type. The quiz is designed to be engaging, shareable, and capture email leads.

## User Workflow

1. **Start Screen** - User lands on quiz page, sees intro with "Start Quiz" button
2. **Questions (1-7)** - User answers each question by clicking an option
3. **Results** - Personality type revealed with description, strengths, weaknesses
4. **Email Capture** - Optional email signup for tips
5. **Share** - Share result on Twitter/X, LinkedIn, or copy link
6. **Retake** - Option to retake quiz (returns to start screen)

## Features

### Scoring System
- Each answer awards points to different personality types
- Points are weighted (1-3 points per type per answer)
- Final personality is determined by highest total score

### Supabase Integration
- Quiz results saved via `submit_quiz_result` RPC
- Email capture via `capture_quiz_email` RPC
- Share tracking via `create_quiz_share` RPC
- Referral click tracking via `increment_share_click` RPC

### UTM Tracking
Captures these URL parameters:
- `utm_source`
- `utm_medium`
- `utm_campaign`
- `ref` (referral code)

### Sharing
- Twitter/X - Opens share dialog with personality result
- LinkedIn - Opens share dialog
- Copy link - Copies referral link to clipboard (with visual feedback)

### Email Capture
After results, users can opt-in to receive:
- Weekly tips tailored to their personality
- Early access to new features

### Shared Result Links
When someone visits via a shared link (`?result=optimizer`), they see:
- The sharer's personality result
- "Take the Quiz" button to discover their own type

## Technical Stack
- Tailwind CSS (via CDN with custom config)
- Supabase JS SDK v2
- Google Fonts (Plus Jakarta Sans, Inter)
- Material Icons Round

## URLs
- Quiz: `https://lunchnear.me/quiz.html`
- Shared result: `https://lunchnear.me/quiz.html?result={type}`
- Referral link: `https://lunchnear.me/quiz.html?ref={code}`

## Database Schema

### quiz_results
- `id` (UUID)
- `personality_type` (text)
- `answers` (JSONB)
- `scores` (JSONB)
- `email` (text, nullable)
- `source`, `medium`, `campaign` (UTM tracking)
- `referrer_code` (text, nullable)
- `user_agent` (text)
- `created_at` (timestamp)

### quiz_shares
- `id` (UUID)
- `result_id` (FK to quiz_results)
- `platform` (twitter/linkedin/copy)
- `referral_code` (unique code for tracking)
- `click_count` (integer)
- `created_at` (timestamp)

## Key JavaScript Functions

| Function | Description |
|----------|-------------|
| `startQuiz()` | Hides start screen, shows questions, renders first question |
| `renderQuestion()` | Displays current question with answer options |
| `selectOption(index)` | Handles answer selection, adds points, advances to next |
| `showResults()` | Calculates winner, saves to Supabase, displays results |
| `retakeQuiz()` | Resets state, returns to start screen |
| `shareTwitter(type)` | Opens Twitter share dialog with referral link |
| `shareLinkedIn(type)` | Opens LinkedIn share dialog |
| `copyLink(type)` | Copies referral link to clipboard |
| `handleEmailSubmit(e, type)` | Saves email to Supabase |
| `checkSharedResult()` | Checks URL for shared result, displays if found |
